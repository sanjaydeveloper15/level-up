Usage of Var, Let & Const

Var Usage in JS

var a = "Hello JS Variable";
document.write(a);

Let Usage in JS

let b = "Hello JS Variable via LET";
document.write(b);
Main thing in this we cannot use let again for same variable assign. In other Words we cannot redclare Variable we can assign value in it if we want.
If we let keyword using for same variable
Uncaught SyntaxError: Identifier 'b' has already been declared

Const Usage in JS

Once assign then you cannot override this variable.
const key = "sf43344ddd";
document.write(key);
